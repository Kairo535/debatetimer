使用说明
保存辩论赛计时器.html文件，双击用Chrome/Edge/Firefox等现代浏览器打开

首次启动弹出使用教程，按指引填写队伍信息、选择赛制，点击「开始辩论」进入计时界面

计时页面通过底部按钮控制计时流程，自由辩论环节点击「切换发言方」按钮操作

右上角可切换主题、语言，打开设置可导出/导入赛事配置、查看历史记录

注意事项
工具纯本地运行，无需联网，所有数据仅存储在本地浏览器，不会上传云端


建议使用现代浏览器打开，老旧浏览器可能存在兼容性问题
